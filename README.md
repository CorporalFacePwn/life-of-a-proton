# Life of a Proton – Manim Project

A pedagogical animation that follows the life of a proton from the
early universe (quark–gluon plasma) through its later cosmic roles.

## Structure

```text
src/
  common/          # shared configs, utilities, diagrams

  ...

  PHASES_OVERVIEW.md
scripts/

environment.yml
